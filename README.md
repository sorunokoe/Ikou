## Features
- [x] Profile: status, private/public, last online
- [x] Friends list
- [x] Owned games
- [x] Dark mode

## Requirements

- iOS 10.0+
- Xcode 10.2.1

## Dependencies
#### Color
- Hue
#### Keyboard
- IQKeyboardManagerSwift
#### Storage
- KeychainSwift
#### Network
- Moya
#### Database
- Realm
#### AutoLayout
- SnapKit
